This IPv4 Address Practice tool spits out random IP Addresses along with its corresponding Subnet Mask in CIDR Notation. You're job my friend, is to compute the Subnet Mask, Network Address, Default Gateway and Broadcast Address and key it into the fields!
Check - Checks whether your input is correct/incorrect.
Answers - Shows the Answer.
Next One - Spits out the next IP Address!

*This tool is in beta phase, in case you experience any bugs/errors or have any suggestions/feedback, please forward it to ranjanmo(at)buffalo.edu
